Configure Hotkey:

To configure the hotkey used by the extension to duplicate the active tab, navigate to the extensions shortut page at chrome://extensions/shortcuts. By default, the extension uses Command+Shift+E on MacOS and Linux or Ctrl+Shift+E on Windows.

About:

Duplicate Tab Hotkey is developed as an open-source application. Source code can be viewed on GitHub, contributions are welcome. Currently the application is maintained by Elijah Verdoorn.
